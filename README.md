# 任务负荷热力图

飞书多维表格仪表盘插件原型。插件会读取用户选择的数据表和字段，把周期型任务按日期区间展开，将计算字段平均分摊到每个自然日或工作日，再按天或按周聚合成热力图。

## 功能

- 动态选择数据表、开始日期、结束日期、计算字段、标题、状态、负责人、分组字段。
- 支持自然日统计和仅工作日统计。
- 支持按天、按周粒度展示，按周时先按天分摊再汇总到周一至周日。
- 支持起始月份加显示月份数，或自定义开始/结束日期。
- 支持小数颜色区间、图例开关、状态/负责人/分组筛选。
- 支持 tooltip 和点击右侧抽屉明细。
- 支持异常数据统计与明细，包括开始日期为空、结束日期早于开始日期、区间内没有可统计日期。

## 本地调试

```bash
npm install
npm run dev
```

默认访问 `http://127.0.0.1:3102/`。

在飞书多维表格插件运行环境中，项目会优先使用 `@lark-base-open/js-sdk` 读取当前 base 的表、字段和记录。本地普通浏览器没有飞书 SDK 上下文时，会自动使用内置 mock 数据，方便验证热力图和配置交互。

## 构建

```bash
npm run build
```

构建产物位于 `dist/`。实际发布到飞书时，请按多维表格仪表盘插件开发指南配置插件入口 URL 或上传构建产物。

## 核心算法

1. 读取记录的开始日期、结束日期和计算字段。
2. 结束日期为空时默认等于开始日期。
3. 开始日期为空、结束日期无效或结束日期早于开始日期时，记录为异常且不参与计算。
4. 根据统计方式生成自然日列表或工作日列表。
5. 每日分摊值等于计算字段数值除以可统计日期数量。
6. 按天模式直接累加每日分摊值。
7. 按周模式先完成每日分摊，再汇总到对应周一至周日。
